                                    // (1) æ¯æ¥ä¸€ä¸ªæ–°è¿æ¥ï¼Œä¸éœ€è¦åˆ›å»ºä¸€ä¸ªçº¿ç¨‹ï¼Œè€Œæ˜¯ç›´æ¥æ³¨å†Œåˆ°clientSelector

                                    keyIterator.remove();
                                }
                            }

                        }
                    }
                }

            }
